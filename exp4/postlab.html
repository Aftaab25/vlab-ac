PostLab Quiz and Exercise

Quiz

1. Op-Amp is abbreviated as ________.

Answer

1. Operational Amplifier

2. The potential output of Op-Amp is _____ times greater than the potential difference of input.

Answer

3. 100,000

3. Op-Amp performs which type of mathematical type operations.

Answer

4. All of the above

4. Op-Amp with negative input type configuration -V is called _______.

Answer

2. Inverting type input

5. An op-amp with negative feedback provides _____ output parameter.

Answer

4. All the above

6. Op-Amp has how many terminals?

Answer

4. 5
